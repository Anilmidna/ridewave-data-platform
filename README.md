  ## About RideWave
  RideWave is a fast-growing ride-hailing and logistics startup
  operating across Mumbai, Delhi, Bengaluru, Chennai, Pune and Hyderabad.
  Founded in 2023, RideWave processes 1000+ rides daily across 6 cities.
  All data was previously managed in Excel sheets and a basic MySQL database.

  ## Platform Architecture
  AWS S3 → Glue → Athena → Redshift (AWS Track)
  AWS S3 → Databricks Delta Lake → Bronze → Silver → Gold (Databricks Track)

  ## Tech Stack

  Redshift Serverless, Databricks, Delta Lake, GitHub
